# 🪨 Stone Wala App

A modern mobile application built using **React Native (Expo)** that connects users with stone vendors and businesses. The app provides a seamless experience for browsing categories, exploring firms, and managing vendor dashboards.

---

## 🚀 Features

### 👤 Guest/User

* Browse stone categories
* View firm listings
* Explore firm details
* Clean and modern UI/UX
* Fast navigation using Expo Router

### 🧑‍💼 Vendor

* Vendor dashboard (UI-first approach)
* Manage products/services
* View inquiries (planned)
* Business profile management (planned)

### 🔐 Authentication (Planned / In Progress)

* User login/signup
* Vendor login/signup
* Role-based navigation (Guest / Vendor)

---

## 🛠️ Tech Stack

* **Frontend:** React Native (Expo)
* **Styling:** NativeWind (Tailwind CSS for React Native)
* **Navigation:** Expo Router
* **State Management:** React Context API
* **Backend (Planned):** Next.js API + MySQL
* **Database ORM (Planned):** Prisma

---

## 📱 App Structure

```
app/
 ├── (auth)            # Authentication screens
 ├── (guest)           # Guest user flow
 ├── (vendor)          # Vendor dashboard & features
 ├── components/       # Reusable UI components
 ├── context/          # Global state management
 └── utils/            # Helper functions
```

---

## 🎯 Development Approach

This project follows a **UI-first development strategy**:

1. Design complete UI screens
2. Implement navigation flow
3. Add state management
4. Integrate backend APIs

---

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/your-username/stone-wala-app.git

# Navigate into the project
cd stone-wala-app

# Install dependencies
npm install

# Start the development server
npx expo start
```
